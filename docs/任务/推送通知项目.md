# 推送通知系统

推送通知系统是使用前后端分离，通过RESTful API实现的。前端使用vue3框架，后端使用golang gin框架。当用户提交代码到代码仓库时，系统进行CODEVIEW然后推送修改变动和 CODEVIEW结果。

## 系统架构

推送通知系统由前端和后端两部分组成。前端使用vue3框架，后端使用golang gin框架。前端负责用户界面和交互，后端负责数据处理和逻辑实现。

## 系统功能

推送通知系统的主要功能包括：

1. 仓库管理
2. 推送目标管理（钉钉、邮箱）
3. 推送内容查询
4. AI模型管理
5. 用户管理
6. 日志管理
7. 推送消息模板管理
8. 提示词管理（用于CODEVIEW提示词、推送消息模板提示词）
